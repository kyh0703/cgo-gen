# Context Docs Migrated

The repository documentation source of truth moved from `.context/` to `docs/`.

Use:

- `docs/README.md` for the documentation map
- `docs/ARCHITECTURE.md` for architecture
- `docs/PRODUCT.md` for product framing
- `docs/roadmaps/current-roadmap.md` for the roadmap
- `docs/references/` for CLI, config, IR, limitations, and hardening notes
- `docs/design-docs/` for longer-lived design docs
- `docs/status/` for status snapshots
